Featured Project

FTC Robotics — Spare Parts #11848

Programming Lead & Software Architect · 5 Seasons (2021–2026)

Lead all robot software development in Java for FTC Team 11848, a community-based team from Lake Villa, Illinois. Designed and tuned PID control systems for drivetrain, arm, and linear slide mechanisms using encoder feedback. Built autonomous routines using Roadrunner odometry localization, sensor fusion, and TensorFlow object detection.

Implemented 3-wheel dead-wheel odometry — computing wheel displacement (d = ticks/tpr × 2πr), orientation change (Δθ = (D_R − D_L) / track), forward/strafe movement, and global position updates via sin/cos rotation matrices. Created motor speed limiters, touch sensor encoder resets, LED match timer alerts, and dual-driver gamepad mappings for two-controller operation.

Built and CAD'd 4 robots across 5 seasons, including "Voyager" and "Nautilus" for the 2024–25 Into The Deep season — featuring mecanum drivetrain, tilting linear slides, 3D-printed 4-DOF arm, flipping specimen claw, and bucket dumping mechanism. Mentored new programmers in Java and control theory. Led 10,000+ outreach hours impacting 100,000+ people across Maker Faires, library STEAM workshops, FLL mentoring, sponsor visits, and international collaboration with Libyan FTC team Hydra #22947.
